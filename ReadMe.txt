Ползунок слева изменяет количество выходящих из каждого заряда линий.
Чем больше линий, тем дольше работает программа, при большом количестве зарядов это играет большую роль.

Левая кнопка мыши добавляет положительный заряд, а правая отрицательный.
Перерисовка происходит каждое нажатие, так что если много раз нажать оно не будет прерывать предыдущие перерисовки и умрёт.
